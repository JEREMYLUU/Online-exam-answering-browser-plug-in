搜题答案助手 - 功能测试页面

测试插件各项功能是否正常工作

📷 截图识题功能测试

测试步骤：
  1. 确保已安装并启用搜题答案助手插件
  2. 配置支持图片的API模型（如GPT-4 Vision、Claude 3 Haiku、Gemini Pro Vision等）
  3. 在页面右上角应该能看到红色的"📷 截图识题"按钮
  4. 点击按钮测试截图功能
注意：截图功能需要选择支持图片输入的API模型，模型名称后会有📷图标标识。

🖱️ 文本选择功能测试

测试步骤：
  1. 选中下面的题目文本
  2. 应该出现蓝色的浮动按钮
  3. 点击按钮获取答案
数学题： 已知函数 f(x) = x² + 2x + 1，求 f(x) 的最小值。
物理题： 一个物体从静止开始做匀加速直线运动，加速度为 2 m/s²，求 5 秒后的速度。
化学题： 写出硫酸（H₂SO₄）与氢氧化钠（NaOH）反应的化学方程式。

⌨️ 快捷键功能测试

测试步骤：
  1. 选中任意题目文本
  2. 按 Ctrl + Shift + Q 快捷键
  3. 应该直接获取答案，无需点击按钮

🔧 插件功能检查

  • 插件图标是否显示在浏览器工具栏
  • 点击插件图标是否弹出设置窗口
  • API配置是否正常工作
  • 模型列表是否动态加载
  • 答案显示框是否正常显示和关闭
  • 错误提示是否清晰明确

⚠️ 常见问题检查

如果功能不正常，请检查：
  • 插件是否已启用（chrome://extensions/）
  • API Key 是否正确配置
  • 网络连接是否正常
  • 是否选择了支持图片的模型（截图功能）
  • 浏览器是否支持（Chrome 88+）
