# Chrome插件发布指南

## 📋 发布前准备

### 1. 生成插件图标
1. 打开 `create_icons.html` 文件
2. 点击"生成所有尺寸图标"按钮
3. 下载生成的图标文件到 `icons/` 目录
4. 确保有以下图标文件：
   - `icons/icon16.png`
   - `icons/icon32.png`
   - `icons/icon48.png`
   - `icons/icon128.png`

### 2. 安装依赖
```bash
npm install
```

### 3. 验证插件
1. 在Chrome中打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择项目文件夹
5. 测试所有功能是否正常

## 🚀 打包插件

### 方法一：使用自动化脚本（推荐）
```bash
npm run build
```

### 方法二：手动打包
1. 选择所有插件文件
2. 右键选择"发送到" → "压缩(zipped)文件夹"
3. 重命名为 `搜题答案助手-v2.0.0.zip`

## 📦 发布到Chrome Web Store

### 1. 注册开发者账户
1. 访问 [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole/)
2. 使用Google账户登录
3. 支付一次性注册费（$5.00 USD）

### 2. 创建新项目
1. 点击"添加新项目"
2. 选择"Chrome扩展程序"
3. 上传打包好的ZIP文件

### 3. 填写插件信息

#### 基本信息
- **插件名称**：搜题答案助手
- **简短描述**：智能识别网页题目，支持多种AI模型获取准确答案
- **详细描述**：
```
搜题答案助手是一款智能的Chrome浏览器插件，能够自动识别网页中的题目，并通过多种AI大模型获取准确详细的答案。

主要功能：
🔍 智能题目识别 - 自动检测网页中的题目内容
🖱️ 鼠标选择 - 选中题目文本后点击浮动按钮获取答案
🔍 一键识别 - 点击右上角"识别题目"按钮自动识别页面题目
🤖 多AI模型支持 - 支持Novita AI、OpenAI、Anthropic Claude、Google Gemini等多种AI模型
⚙️ 灵活配置 - 支持API Key配置、模型选择、参数调整
⌨️ 快捷键支持 - Ctrl+Shift+Q 快速获取答案

支持的AI模型：
• Novita AI - Minimax M1系列模型
• OpenAI - GPT-4、GPT-4 Turbo、GPT-3.5 Turbo
• Anthropic Claude - Claude 3系列模型
• Google Gemini - Gemini Pro系列
• 自定义API - 支持自定义端点和模型

使用简单，功能强大，是学习和工作的得力助手！
```

#### 分类信息
- **类别**：生产力
- **语言**：中文（简体）
- **目标用户**：学生、教师、研究人员

#### 隐私政策
- **隐私政策URL**：https://your-domain.com/privacy-policy
- **数据处理**：插件不会收集或存储用户个人信息

### 4. 上传资源

#### 截图要求
- **尺寸**：1280x800 像素
- **格式**：PNG
- **数量**：至少1张，最多5张
- **内容建议**：
  - 插件主界面截图
  - 题目识别功能演示
  - 答案生成效果展示
  - 设置界面截图

#### 图标要求
- **尺寸**：128x128 像素
- **格式**：PNG
- **背景**：透明或白色

### 5. 设置发布选项

#### 可见性
- **公开**：所有人都可以在Chrome Web Store中找到
- **私有**：只有您和您邀请的用户可以安装

#### 发布地区
- 选择"全球"或特定地区

#### 年龄限制
- 选择"适合所有年龄"

### 6. 提交审核
1. 检查所有信息是否正确
2. 点击"提交审核"
3. 等待Google审核（通常1-3个工作日）

## 🔄 更新插件

### 1. 更新版本号
在 `manifest.json` 中更新版本号：
```json
{
  "version": "2.0.1"
}
```

### 2. 重新打包
```bash
npm run build
```

### 3. 上传新版本
1. 在Developer Dashboard中找到您的插件
2. 点击"上传新版本"
3. 上传新的ZIP文件
4. 填写更新说明
5. 提交审核

## 🚨 常见问题

### 审核被拒
**原因**：功能描述不准确
**解决**：修改描述，确保与实际功能一致

**原因**：截图质量差
**解决**：重新制作高质量截图

**原因**：权限过多
**解决**：检查并移除不必要的权限

### 安装失败
**原因**：文件损坏
**解决**：重新打包并上传

**原因**：版本兼容性
**解决**：检查Chrome版本要求

### 功能异常
**原因**：API配置错误
**解决**：检查API Key和端点设置

**原因**：权限不足
**解决**：确保所有必要权限已申请
