# 搜题答案助手 - 安装指南

## 快速安装

### 步骤1：在Chrome中安装插件

1. **打开Chrome扩展管理页面**
   - 在Chrome地址栏输入：`chrome://extensions/`
   - 或者点击菜单 → 更多工具 → 扩展程序

2. **开启开发者模式**
   - 在页面右上角找到"开发者模式"开关
   - 点击开启（开关会变成蓝色）

3. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 在弹出的文件选择器中，选择整个项目文件夹
   - 点击"选择文件夹"

4. **验证安装**
   - 插件应该出现在扩展程序列表中
   - 浏览器工具栏中应该出现插件图标

### 步骤2：配置API Key

1. **获取Novita AI API Key**
   - 访问 https://novita.ai/
   - 注册账号并获取API Key

2. **配置插件**
   - 点击浏览器工具栏中的插件图标
   - 在弹出窗口中输入你的API Key
   - 点击"保存设置"
   - 点击"测试API连接"验证配置

## 使用方法

### 基本使用

#### 方法一：一键自动识别（推荐）
1. **找到识别按钮**
   - 页面加载完成后，右上角会出现绿色的"识别题目"按钮
   - 按钮带有🔍图标，易于识别

2. **自动识别题目**
   - 点击"识别题目"按钮
   - 插件会自动扫描页面内容，寻找题目
   - 如果找到题目，会自动获取答案

#### 方法二：手动选择题目
1. **选择题目文本**
   - 在网页上用鼠标选中题目内容
   - 会出现蓝色的浮动按钮

2. **获取答案**
   - 点击浮动按钮
   - 等待AI生成答案
   - 答案会在页面右上角显示

### 高级功能

- **快捷键**：按 `Ctrl+Shift+Q` 快速获取答案
- **自动识别**：在设置中开启"自动识别题目"
- **浮动按钮**：可以关闭浮动按钮显示

## 功能说明

### 自动识别按钮
- **位置**：页面右上角固定位置
- **样式**：绿色渐变背景，带有🔍图标
- **功能**：一键识别页面中的所有题目
- **显示时机**：页面加载完成后1秒自动显示

### 浮动按钮
- **触发条件**：选中文本后自动出现
- **样式**：蓝色圆形按钮，带有"?"符号
- **功能**：获取选中文本的答案

### 答案显示框
- **位置**：页面右上角
- **功能**：显示题目和AI生成的答案
- **关闭**：点击右上角的"×"按钮

## 故障排除

### 插件无法加载

**问题**：点击"加载已解压的扩展程序"后没有反应

**解决方案**：
1. 检查Chrome版本是否支持Manifest V3（需要Chrome 88+）
2. 确认所有必需文件都存在
3. 检查manifest.json文件格式是否正确

### API调用失败

**问题**：点击获取答案时显示错误

**解决方案**：
1. 检查API Key是否正确
2. 确认网络连接正常
3. 验证Novita AI服务状态
4. 检查API调用频率是否超限

### 题目识别不准确

**问题**：插件无法正确识别题目

**解决方案**：
1. 尝试手动选择题目文本
2. 确保题目文本长度在10-1000字符之间
3. 检查题目是否包含关键词（如"题目"、"问题"、"计算"等）

### 自动识别按钮不显示

**问题**：页面右上角没有看到"识别题目"按钮

**解决方案**：
1. 刷新页面，按钮会在1秒后自动显示
2. 检查是否有其他插件冲突
3. 确认插件已正确加载
4. 检查浏览器控制台是否有错误信息

## 更新插件

当插件代码更新时：

1. 在 `chrome://extensions/` 页面找到插件
2. 点击插件卡片上的刷新图标
3. 或者点击"重新加载"按钮

## 卸载插件

1. 在 `chrome://extensions/` 页面找到插件
2. 点击"移除"按钮
3. 确认卸载

## 技术支持

如果遇到其他问题，请：

1. 检查Chrome控制台是否有错误信息
2. 查看插件的错误日志
3. 联系开发者获取支持
