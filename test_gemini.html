🔧 Gemini API 连接测试工具

📋 使用步骤

  1. 获取您的 Gemini API Key（访问 Google AI Studio）
  2. 将 API Key 粘贴到下面的输入框中
  3. 点击"测试网络连接"检查基本连接
  4. 点击"测试API路径"验证API端点格式
  5. 点击"获取模型列表"加载最新的可用模型
  6. 从下拉框中选择要测试的模型
  7. 点击"验证模型可用性"按钮检查哪些模型可用
  8. 输入测试消息，可选择包含题目的图片文件（仅支持图片的模型）
  9. 点击"测试文本生成"或"测试图片答题"按钮
  10. 查看结果和错误信息

📷 支持图片的模型：gemini-2.5-flash, gemini-2.5-pro, gemini-1.5-flash, gemini-1.5-pro, gemini-2.0系列等

🎯 图片答题功能：AI会分析图片中的题目并给出详细解答，包括选择题原因、计算题步骤、问答题答案等

点击"获取模型列表"按钮来加载最新的可用模型
选择包含题目的图片文件，AI会分析并给出详细解答
