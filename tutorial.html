API Key 申请教程

免费额度和免费模型会随平台政策变化，最终以各平台控制台为准。插件会尽量显示平台返回的全部模型列表，不替用户判断模型类型；截图能否成功以官网模型能力说明和实际 API 返回为准。

⚙️ 插件里为什么有两套 API 配置？

文文字搜题 API

用于“选中文字后点击 ?”。这里可以使用便宜或免费的文本模型，例如 DeepSeek、Qwen、Groq、Kimi、Mistral Small 等。

图截图识题 API

用于页面右上角“一键截图识题”。插件会显示平台返回的模型列表，也支持手动填写官网模型 Code；如果模型不支持图片，平台 API 会返回错误。

推荐新手配置：文字搜题选 DeepSeek 或 Groq；截图识题可先试 Google Gemini。官网“免费额度/全部模型”页面里的模型 Code 都可以手动填写；是否适合截图，以官网对该模型的能力说明为准。

🆓 免费额度优先推荐

1Google Gemini

首选：官方提供 Free Tier，适合一键截图识题。插件里选择 Google Gemini (免费额度 / 截图优先)，模型优先选 Gemini Flash / Flash Lite。

2OpenRouter

备选：可以使用免费模型或 openrouter/free 路由，一个 Key 可以试多家模型。免费模型可能限流，稳定性取决于平台当前可用模型。

3SiliconFlow 硅基流动

国内友好：提供免费模型，适合先试 DeepSeek/Qwen；截图识题可按官网模型能力选择对应模型 Code。

4阿里云百炼 / 通义千问

新人友好：新用户通常有免费额度，国内访问稳定。官网页面显示的模型 Code 可以直接填入插件。

5Groq / Mistral

补充选择：Groq 适合高速文本搜题；Mistral 的 Free mode 可试 Mistral/Pixtral。若要截图识题，请确认当前模型支持图片。

🚀 DeepSeek (深度求索) 付费低成本，适合文本题

1注册账号

访问 DeepSeek 开放平台，注册或登录账号。

前往 DeepSeek 平台
2创建 API Key

在左侧菜单找到“API Keys”，点击“创建 API Key”。

3配置插件

复制生成的 Key (以 sk- 开头)。

在插件中选择 DeepSeek (深度求索)，配置如下：

  • API端点: https://api.deepseek.com/chat/completions
  • 模型: deepseek-chat (V3) 或 deepseek-reasoner (R1)

🆓 OpenRouter 免费模型 / 一个 Key 使用多家模型

1注册账号

访问 OpenRouter，登录后进入 Keys 页面。

前往 OpenRouter Keys
2创建 API Key

点击创建 Key，复制生成的密钥。

3配置插件

在插件中选择 OpenRouter (聚合平台)。

  • API端点: https://openrouter.ai/api/v1/chat/completions
  • 模型: 可先选 openrouter/free 或下拉框里的免费模型；截图识题是否可用以 OpenRouter 模型说明为准。

🆓 Google Gemini 免费额度，截图首选

1访问 AI Studio

你需要一个 Google 账号。

前往 Google AI Studio
2创建 Key

点击 "Create API key"，选择一个 Google Cloud 项目（如果没有会自动创建）。

3配置插件

在插件中选择 Google Gemini (免费额度 / 截图优先) 并粘贴 API Key。

  • API端点: https://generativelanguage.googleapis.com/v1beta/models/gemini-1.5-flash:generateContent
  • 模型: 可直接填写 Google AI Studio / Gemini 文档显示的模型 Code。
推荐理由： Gemini API 提供 Free Tier，Gemini 系列多模态（识图）能力强，适合截图识题。
注意：Google API 在某些地区可能需要网络工具才能访问。
