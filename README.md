# 搜题答案助手 Chrome 插件

一个智能的Chrome浏览器插件，能够通过截图识别题目，并通过多种AI大模型获取答案。

## 功能特点

- 📷 **截图识题**：点击右上角"截图识题"按钮，截取屏幕区域识别题目
- 🖱️ **鼠标选择**：选中题目文本后点击浮动按钮获取答案
- 🤖 **多AI模型支持**：支持Novita AI、OpenAI、Anthropic Claude、Google Gemini等多种AI模型
- 🖼️ **图片识别**：支持图片输入，可识别截图中的题目内容
- 🔄 **动态模型加载**：自动从API获取最新的模型列表，无需手动更新
- ⚙️ **灵活配置**：支持API Key配置、模型选择、参数调整
- ⌨️ **快捷键支持**：Ctrl+Shift+Q 快速获取答案
- 📱 **响应式设计**：适配不同屏幕尺寸

## 安装步骤

### 1. 下载插件文件
将项目文件夹下载到本地。

### 2. 在Chrome中加载插件
1. 打开Chrome浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

### 3. 配置API设置
1. 点击浏览器工具栏中的插件图标
2. 选择您要使用的AI服务提供商
3. 输入对应的API Key
4. 插件会自动从API获取可用的模型列表
5. 选择合适的模型和参数
6. 点击"保存设置"
7. 点击"测试API连接"验证配置

## 使用方法

### 方法一：截图识题（推荐）
1. 在网页右上角找到红色的"📷 截图识题"按钮
2. 点击按钮，选择要截图的区域
3. 插件会自动分析截图中的题目并生成答案

### 方法二：鼠标选择题目
1. 在网页上选中题目文本
2. 点击出现的蓝色浮动按钮
3. 等待AI生成答案

### 方法三：快捷键操作
- 按 `Ctrl+Shift+Q` 快速获取当前选中文本的答案

## API配置说明

### 支持图片的API

#### OpenAI
- **获取API Key**：[访问OpenAI平台](https://platform.openai.com/api-keys) → 登录 → API Keys → Create new secret key
- **推荐模型**：GPT-4 Vision（支持图片）或 GPT-4o（最新多模态）

#### Anthropic Claude
- **获取API Key**：[访问Anthropic控制台](https://console.anthropic.com/) → 登录 → API Keys → Create Key
- **推荐模型**：Claude 3 Haiku（快速图片识别）或 Claude 3 Sonnet（平衡性能）

#### Google Gemini
- **获取API Key**：[访问Google AI Studio](https://makersuite.google.com/app/apikey) → 登录 → Get API key
- **推荐模型**：Gemini Pro Vision（专门支持图片）

#### Novita AI
- **获取API Key**：[访问Novita AI官网](https://novita.ai/) → 注册/登录 → API管理 → 创建新的API Key
- **推荐模型**：Minimax M1-80K（支持长文本和图片）

### 仅支持文本的API

#### OpenAI
- **获取API Key**：[访问OpenAI平台](https://platform.openai.com/api-keys) → 登录 → API Keys → Create new secret key
- **推荐模型**：GPT-4（最强大）或 GPT-3.5 Turbo（性价比高）

## 参数配置

- **最大Token数**：控制答案的最大长度（100-100000）
- **温度 (创造性)**：0=保守准确，2=创造性回答
- **显示浮动图标**：开启后选中文本时显示浮动按钮

## 界面元素说明

### 截图按钮
- **位置**：页面右上角固定位置
- **样式**：红色渐变背景，带有📷图标
- **功能**：启动截图功能，选择屏幕区域进行题目识别
- **显示时机**：页面加载完成后1秒自动显示

### 浮动按钮
- **触发条件**：选中文本后自动出现
- **样式**：蓝色圆形按钮，带有"?"符号
- **功能**：获取选中文本的答案

### 答案显示框
- **位置**：页面右上角
- **功能**：显示题目和AI生成的答案
- **关闭方式**：
  - 点击右上角的"×"按钮
  - 按ESC键
  - 点击答案框外的任意区域

## 注意事项

1. **API Key安全**：请妥善保管您的API Key，不要泄露给他人
2. **图片支持**：不是所有API模型都支持图片输入，请选择支持图片的模型
3. **使用频率**：注意API调用频率限制，避免过度使用
4. **网络连接**：确保网络连接稳定，以便正常调用API
5. **浏览器兼容**：仅支持Chrome浏览器及基于Chromium的浏览器
6. **模型选择**：根据题目复杂度和成本考虑选择合适的模型

## 故障排除

### Gemini API 连接问题

如果您遇到 Gemini API 连接问题，请参考详细的故障排除指南：

📖 **[Gemini API 故障排除指南](GEMINI_TROUBLESHOOTING.md)**

常见问题包括：
- API Key 无效或已过期
- 网络连接问题
- 模型名称格式错误
- 请求格式问题

### 通用故障排除

1. **扩展无法加载**：检查浏览器扩展管理页面，确保扩展已启用
2. **权限问题**：确保扩展有截图和访问网页内容的权限
3. **API 调用失败**：检查 API Key 和网络连接
4. **模型加载失败**：检查 API 配置和模型名称
