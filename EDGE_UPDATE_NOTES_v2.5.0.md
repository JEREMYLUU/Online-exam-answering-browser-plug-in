# 搜题答案助手 v2.5.0 更新说明

## 本次更新

- 新增完整设置中心，可集中配置文字搜题 API、截图识题 API、使用偏好、账户套餐预留和隐私说明。
- 新增首次安装引导，安装插件后自动打开快速开始页面，帮助用户选择免费额度平台并完成配置。
- 弹窗新增“设置”入口，用户可从小弹窗进入完整设置页。
- 设置中心支持获取平台模型列表、手动填写官网模型 Code、测试 API 连接。
- 保留现有本地存储字段，兼容旧版本配置。

## 商业化预留

- 预留免费试用、9.9 元基础版、59 元/月 Pro 版展示区。
- 当前版本未接入登录、支付、服务端额度校验；后续如启用收费，应通过后端校验会员和用量。

## 隐私说明

- API Key 和设置保存在浏览器扩展存储中。
- 只有用户选中文字或点击截图按钮后，插件才会向用户配置的 AI 平台发起请求。
- 插件不内置远程 JavaScript，不通过开发者服务器收集用户数据。
