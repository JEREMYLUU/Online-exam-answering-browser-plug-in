# 搜题答案助手 v2.4.0 更新说明

## 面向用户的更新说明

本次更新重点优化了模型配置、免费模型使用和响应速度：

- 新增文字搜题 API 与截图识题 API 分开配置。
- 文字搜题可使用 DeepSeek、Qwen、Groq、Kimi 等文本模型。
- 截图识题可单独配置 Gemini、Qwen-VL、Pixtral、GPT-4o 等支持图片输入的模型。
- 模型列表改为按平台接口全量显示，不再由插件自行筛选文本模型或多模态模型。
- 新增手动填写模型 Code，官网看到的模型 Code 可直接填入插件。
- 优先展示带免费额度或免费模型的平台，例如 Gemini、OpenRouter、SiliconFlow、阿里云百炼、Groq、Mistral。
- 新增快速模式，默认开启，优先输出答案并减少长篇解析。
- 优化截图识题速度，截图会自动压缩后再发送，减少上传和模型处理时间。
- 降低默认输出长度，减少等待时间和 API 成本。
- 优化 API Key 教程，说明文字模型和截图模型的区别。

## 推荐配置

- 文字搜题 API：DeepSeek、Qwen、Groq、Kimi、Mistral Small 等文本模型。
- 截图识题 API：Gemini、Qwen-VL、Pixtral、GPT-4o、Claude 视觉模型等支持图片输入的模型。
- 如果平台模型列表和官网不一致，可在“官网模型 Code”输入框手动填写。

## 给审核人员的说明

- 本扩展为浏览器内的学习辅助工具，用户需自行配置第三方 AI 平台 API Key。
- API Key 保存在浏览器扩展存储中，用于向用户选择的第三方模型平台发起请求。
- 扩展不会内置、收集或上传 API Key 到开发者服务器。
- 截图识题仅在用户点击页面中的“截图识题”按钮后触发。
- 选中文字搜题仅在用户选中文本并点击浮动按钮后触发。
- 本版本保留 Manifest V3 service worker 架构。
